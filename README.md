# arq-dashboard

A dashboard for [ARQ](https://github.com/samuelcolvin/arq) built with [FastAPI](https://github.com/tiangolo/fastapi).

![img](./screenshots/stats.png)
![img](./screenshots/jobs.png)

## Requirements

- Python 3.8+

## Installation

```bash
pip install abuse_whois
```
